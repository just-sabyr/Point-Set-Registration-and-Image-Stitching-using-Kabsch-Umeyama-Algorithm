# Dependencies
Python v3.10.13


# Installation

1. Create a python virtual environment 
```bash
    python3 -m venv .venv
```

2. Activate the environment
```bash
    source .venv/bin/activate
```

3. Install the python libraries
``` bash
    pip install -r requirements.txt
```

4. Run the main.ipynb
